## Herhaling werkwoordstijden

### Présent

Je vervoegt regelmatige werkwoorden door een uitgang aan de stam toe te voegen:

|              | **\-er** | **\-ir**   | **\-re** |
| ------------ | -------- | ---------- | -------- |
| *je*         | *\-e*    | *\-is*     | *\-s*    |
| *tu*         | *\-es*   | *\-is*     | *\-s*    |
| *il/elle/on* | *\-e*    | *\-it*     | *\-*     |
| *nous*       | *\-ons*  | *\-issons* | *\-ons*  |
| *vous*       | *\-ez*   | *\-issez*  | *\-ez*   |
| *ils/elles*  | *\-ent*  | *\-issent* | *\-ent*  |

### Passé Composé

De passé composé maak je met een hulpwerkwoord (*avoir* of *être*) en een voltooid deelwoord.

| **Werkwoord op** | **Voltooid deelwoord** |
| ---------------- | ---------------------- |
| **\-er**         | **\-é**                |
| **\-ir**         | **\-i**                |
| **\-re**         | **\-u**                |

Als je *être* gebruikt als hulpwerkwoord, moet je het voltooid deelwoord laten overeenkomen met het onderwerp (mannelijk/vrouwelijk, enkelvoud/meervoud).

Bij de onregelmatige werkwoorden gebruik je een ander voltooid deelwoord:

| **Werkwoord** | **Voltooid deelwoord** |
| ------------- | ---------------------- |
| *avoir*       | *eu*                   |
| *être*        | *été*                  |
| *faire*       | *fait*                 |
| *prendre*     | *pris*                 |
| *pouvoir*     | *pu*                   |
| *vouloir*     | *voulu*                |
| *voir*        | *vu*                   |
| *devoir*      | *dû*                   |
| *suivre*      | *suivi*                |
| *atteindre*   | *atteint*              |

### Imparfait

Je maakt de imparfait door bij de *nous*-vorm van het présent de uitgang *\-ons* weg te halen en daar een nieuwe uitgang achter te zetten:

|              | **Uitgang** |
| ------------ | ----------- |
| *je*         | **\-ais**   |
| *tu*         | **\-ais**   |
| *il/elle/on* | **\-ait**   |
| *nous*       | **\-ions**  |
| *vous*       | **\-iez**   |
| *ils/elles*  | **\-aient** |

Let op! Bij *être* gebruik je de stam **ét-** (*j'étais*, *tu étais*, *c'était*)

Bij werkwoorden op *\-cer* krijg je een *ç* voor de *a*.
*je commençais*

Bij werkwoorden op *\-ger* krijg je een extra *e* voor de *a*.
*je mangeais*, *nous mangions*

### Futur Simple

Je maakt de futur simple door een uitgang achter het hele werkwoord te zetten. Bij werkwoorden op *\-re* moet je eerst de *\-e* weghalen.

|              | **Uitgang** |
| ------------ | ----------- |
| *je*         | **\-ai**    |
| *tu*         | **\-as**    |
| *il/elle/on* | **\-a**     |
| *nous*       | **\-ons**   |
| *vous*       | **\-ez**    |
| *ils/elles*  | **\-ont**   |


Bij de onregelmatige werkwoorden gebruik je een andere stam:

| **Werkwoord** | **Stam**    |
| ------------- | ----------- |
| *avoir*       | *aur-*      |
| *être*        | *ser-*      |
| *aller*       | *ir-*       |
| *faire*       | *fer-*      |
| *pouvoir*     | *pourr-*    |
| *vouloir*     | *voudr-*    |
| *venir*       | *viendr-*   |
| *voir*        | *verr-*     |
| *devoir*      | *devr-*     |
| *atteindre*   | *atteindr-* |

### Conditionnel

Je maakt de conditionnel door de stam van de futur simple te nemen en daar de uitgangen van de imparfait achter te zetten:

|              | **Uitgang** |
| ------------ | ----------- |
| *je*         | **\-ais**   |
| *tu*         | **\-ais**   |
| *il/elle/on* | **\-ait**   |
| *nous*       | **\-ions**  |
| *vous*       | **\-iez**   |
| *ils/elles*  | **\-aient** |

De onregelmatige stammen zijn dezelfde als bij de futur simple.

## Gérondif

De **gérondif** is een combinatie van *en* en een **tegenwoordig deelwoord**. Het tegenwoordig deelwoord is een werkwoordsvorm die eindigt op *-ant*. Je hoeft deze niet zelf te kunnen vormen.

Je komt de gérondif tegen in de volgende situaties:

- **Gelijktijdigheid**: twee dingen gebeuren tegelijk  
  *Il écoute de la musique en faisant ses devoirs.* $\Rightarrow$ Hij luistert naar muziek terwijl hij zijn huiswerk maakt.
- **Voorwaarde**: iets is nodig om iets anders te bereiken  
  *En travaillant dur, tu réussiras.* $\Rightarrow$ Als je hard werkt, zul je slagen.
- **Manier**: hoe iets gedaan wordt  
  *Elle a répondu en souriant.* $\Rightarrow$ Ze antwoordde door te glimlachen.

## Lidwoorden

Het Frans heeft 3 soorten lidwoorden: **bepaalde lidwoorden** (*le*, *la*, *l'*, *les*), **onbepaalde lidwoorden** (*un*, *une*, *des*) en **delende lidwoorden** (*du*, *de la*, *de l'*, *des*).

Je gebruikt een **delend lidwoord** als je in het Nederlands geen lidwoord gebruikt en je de hoeveelheid niet weet.  
*Je mange du pain.* $\Rightarrow$ Ik eet brood.  
*Elle boit de la limonade.* $\Rightarrow$ Zij drinkt limonade.

Als je praat over dingen in het algemeen, gebruik je een **bepaald lidwoord** in plaats van een delend lidwoord.  
*J'aime le chocolat.* $\Rightarrow$ Ik hou van chocolade. (in het algemeen)

Na een hoeveelheidswoord (*beaucoup*, *peu*, *assez*, etc.) gebruik je *de* of *d'* in plaats van *du*, *de la*, *de l'* of *des*.  
*Je mange beaucoup de pain.* $\Rightarrow$ Ik eet veel brood.  
*Elle a peu d'amis.* $\Rightarrow$ Zij heeft weinig vrienden.

Na een **ontkenning** gebruik je ook *de* of *d'*, behalve bij *être*.  
*Je ne mange pas de viande.* $\Rightarrow$ Ik eet geen vlees.  
*Ce n'est pas du sucre.* $\Rightarrow$ Dit is geen suiker. (met *être*, dus wel delend lidwoord)

Na de werkwoorden *aimer*, *préférer*, *détester*, *adorer*, *supporter* en *haïr* gebruik je altijd een **bepaald lidwoord**, ook na een ontkenning.  
*J'adore les fraises.* $\Rightarrow$ Ik ben dol op aardbeien.  
*Je n'aime pas le café.* $\Rightarrow$ Ik hou niet van koffie.
