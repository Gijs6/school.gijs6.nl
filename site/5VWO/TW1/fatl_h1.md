## Grammatica

### Onregelmatige werkwoorden

|                   | ***commencer*** (beginnen) | ***jeter*** (weggooien) | ***acheter*** (kopen) | ***partager*** (delen) |
| ----------------- | -------------------------- | ----------------------- | --------------------- | ---------------------- |
| **présent**       | *je commence*              | *je jette*              | *j'achète*            | *je partage*           |
|                   | *tu commences*             | *tu jettes*             | *tu achètes*          | *tu partages*          |
|                   | *il/elle/on commence*      | *il/elle/on jette*      | *il/elle/on achète*   | *il/elle/on partage*   |
|                   | *nous commençons*          | *nous jetons*           | *nous achetons*       | *nous partageons*      |
|                   | *vous commencez*           | *vous jetez*            | *vous achetez*        | *vous partagez*        |
|                   | *ils/elles commencent*     | *ils/elles jettent*     | *ils/elles achètent*  | *ils/elles partagent*  |
| **passé composé** | *j'ai commencé*            | *j'ai jeté*             | *j'ai acheté*         | *j'ai partagé*         |
| **imparfait**     | *je commençais*            | *je jetais*             | *j'achetais*          | *je partageais*        |
| **futur simple**  | *je commencerai*           | *je jetterai*           | *j'achèterai*         | *je partagerai*        |
| **conditionnel**  | *je commencerais*          | *je jetterais*          | *j'achèterais*        | *je partagerais*       |

### Herhaling werkwoordstijden

#### Présent

Je vervoegt regelmatige werkwoorden door een uitgang aan de stam toe te voegen:

|              | **\-er** | **\-ir**   | **\-re** |
| ------------ | -------- | ---------- | -------- |
| *je*         | *\-e*    | *\-is*     | *\-s*    |
| *tu*         | *\-es*   | *\-is*     | *\-s*    |
| *il/elle/on* | *\-e*    | *\-it*     | *\-*     |
| *nous*       | *\-ons*  | *\-issons* | *\-ons*  |
| *vous*       | *\-ez*   | *\-issez*  | *\-ez*   |
| *ils/elles*  | *\-ent*  | *\-issent* | *\-ent*  |

#### Passé Composé

De passé composé maak je met een hulpwerkwoord (*avoir* of *être*) en een voltooid deelwoord.

| **Werkwoord op** | **Voltooid deelwoord** |
| ---------------- | ---------------------- |
| **\-er**         | **\-é**                |
| **\-ir**         | **\-i**                |
| **\-re**         | **\-u**                |

Als je *être* gebruikt als hulpwerkwoord, moet je het voltooid deelwoord laten overeenkomen met het onderwerp (mannelijk/vrouwelijk, enkelvoud/meervoud).

Bij de onregelmatige werkwoorden gebruik je een ander voltooid deelwoord:

| **Werkwoord** | **Voltooid deelwoord** | **Werkwoord** | **Voltooid deelwoord** |
| ------------- | ---------------------- | ------------- | ---------------------- |
| *avoir*       | *eu*                   | *pouvoir*     | *pu*                   |
| *être*        | *été*                  | *vouloir*     | *voulu*                |
| *faire*       | *fait*                 | *voir*        | *vu*                   |
| *prendre*     | *pris*                 | *devoir*      | *dû*                   |

#### Imparfait

Je maakt de imparfait door bij de *nous*-vorm van het présent de uitgang *\-ons* weg te halen en daar een nieuwe uitgang achter te zetten:

|              | **Uitgang** |
| ------------ | ----------- |
| *je*         | **\-ais**   |
| *tu*         | **\-ais**   |
| *il/elle/on* | **\-ait**   |
| *nous*       | **\-ions**  |
| *vous*       | **\-iez**   |
| *ils/elles*  | **\-aient** |

Let op! Bij *être* gebruik je de stam **ét-** (*j'étais*, *tu étais*, *c'était*

Bij werkwoorden op *\-cer* krijg je een *ç* voor de *a*.
*je commençais*

Bij werkwoorden op *\-ger* krijg je een extra *e* voor de *a*.
*je mangeais*, *nous mangions*

#### Futur Simple

Je maakt de futur simple door een uitgang achter het hele werkwoord te zetten. Bij werkwoorden op *\-re* moet je eerst de *\-e* weghalen.

|              | **Uitgang** |
| ------------ | ----------- |
| *je*         | **\-ai**    |
| *tu*         | **\-as**    |
| *il/elle/on* | **\-a**     |
| *nous*       | **\-ons**   |
| *vous*       | **\-ez**    |
| *ils/elles*  | **\-ont**   |

Bij zinnen die beginnen met *si* (als), en dus een voorwaarde aangeven, gebruik je verschillende tijden in de hoofd- en bijzin. Als *si* gevolgd wordt door een **présent**, dan gebruik je in de hoofdzin de **futur simple**.

Bij de onregelmatige werkwoorden gebruik je een andere stam:

| **Werkwoord** | **Stam** | **Werkwoord** | **Stam**  |
| ------------- | -------- | ------------- | --------- |
| *avoir*       | *aur-*   | *pouvoir*     | *pourr-*  |
| *être*        | *ser-*   | *vouloir*     | *voudr-*  |
| *aller*       | *ir-*    | *venir*       | *viendr-* |
| *faire*       | *fer-*   | *voir*        | *verr-*   |

#### Conditionnel

Je maakt de conditionnel door de stam van de futur simple te nemen en daar de uitgangen van de imparfait achter te zetten:

|              | **Uitgang** |
| ------------ | ----------- |
| *je*         | **\-ais**   |
| *tu*         | **\-ais**   |
| *il/elle/on* | **\-ait**   |
| *nous*       | **\-ions**  |
| *vous*       | **\-iez**   |
| *ils/elles*  | **\-aient** |

De onregelmatige stammen zijn dezelfde als bij de futur simple.

Bij zinnen die beginnen met *si* (als), en dus een voorwaarde aangeven, gebruik je verschillende tijden in de hoofd- en bijzin. Als *si* gevolgd wordt door een **imparfait**, dan gebruik je in de hoofdzin de **conditionnel**.

## Betrekkelijke voornaamwoorden

Een betrekkelijk voornaamwoord verbindt een betrekkelijke bijzin met een hoofdzin.

Je gebruikt *qui* (die of dat) als het verwijst naar het **onderwerp** in de bijzin. In de bijzin heeft *qui* de functie van onderwerp.

Je gebruikt *que* (die of dat) als het verwijst naar het **lijdend voorwerp** in de bijzin. In de bijzin heeft *que* de functie van lijdend voorwerp.

*Que* wordt *qu'* voor een klinker of een stomme *h*. *Qui* mag je nooit afkorten.

Als je snel weten of je *qui* of *que* moet gebruiken, dan kijk je naar het eerste woord van de bijzin. Is dat een werkwoord? Dan gebruik je *qui*. Staat er een onderwerp? Dan gebruik je *que*.

Een ander betrekkelijk voornaamwoord is *dont*. *Dont* komt altijd voor bij werkwoorden met het voorzetsel *de*, zoals *parler de*, *avoir besoin de* en *avoir envie de*.
