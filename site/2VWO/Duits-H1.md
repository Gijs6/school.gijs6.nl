## Grammatica

### Das Personalpronomen (persoonlijk voornaamwoorden) und das Verb sind (zijn)

| ich bin      | ik ben      |
| ------------ | ----------- |
| du bist      | jij bent    |
| er ist       | hij is      |
| sie ist      | zij is      |
| es ist       | het is      |
| wir sind     | wij zijn    |
| ihr seid     | jullie zijn |
| sie sind     | zij zijn    |
| Sie[^1] sind | u bent      |

[^1]: De beleefdheidsvorm Sie schrijf je altijd met een hoofdletter.
